BusFleetEvaluationToolkit

Author:		Alexander Yoshizumi

Start Date:	07 October 2018
Last Update:	04 December 2018

Description:
This folder contains scripts, data, and explanatory documents used for evaluating
the relative spatial distribution of the tailpipe emissions across GoTriangle's current diesel bus fleet.

Please consult the "UserGuide.pdf" in the documents folder for additional information on how to
navigate and use tools found within this project folder.
